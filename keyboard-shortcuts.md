# 快捷键  

**最近已经上传完毕！** 
   
这篇网页是为了 [Balsamiq Mockups 3](https://balsamiq.com/products/mockups/) 而上传的。原来的文件您可以点击[这里](http://media.balsamiq.com/files/Balsamiq_Mockups_v1-v2_Docs.pdf)来观看。
  
下面这张表列出了在 Balsamiq Mockups 中所有可用的快捷键和修饰符。尽管在桌面版中所有的快捷键都是有效的，但是当你在浏览器中编辑模型时，有些快捷键可能就不管用了。  

下面的快捷键适用于 Mockups 的桌面版 3 及以上版本。你可以下载右面的链接以做备忘。  

## 控件使用

|选中    ||
|:---------|:---------|
|编辑选中内容|  <kbd>Enter</kbd>或<kbd>F2</kbd>|   
|增加选中内容 |  按住<kbd>SHFIT</kbd>键并单击|  
|选中切换      |  按住<kbd>CTRL</kbd>键并单击|  
|全选           |           <kbd>CTRL</kbd> + <kbd>A</kbd>|  
|全部反选        |           <kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>A</kbd>|  
|将/不将选择作为标记|         <kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>M</kbd>|

|大小/位置    |         |  
|:---------|:------------|  
|调整保持纵横比|按住<kbd>SHIFT</kbd>键|  
|移动或调整过程中禁止捕捉|按住<kbd>CTRL</kbd>键|  
|加大字体或图标大小|<kbd>CTRL</kbd> + <kbd>]</kbd>|  
|减小字体或图标大小|<kbd>CTRL</kbd> + <kbd>[</kbd>|  
|选择微调1px|<kbd>↑</kbd><kbd>↓</kbd><kbd>←</kbd><kbd>→</kbd>|  
|选择微调 10px|<kbd>SHIFT</kbd> + <kbd>↑</kbd><kbd>↓</kbd><kbd>←</kbd><kbd>→</kbd>|  
|选择微调大小1px|<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>↑</kbd><kbd>↓</kbd><kbd>←</kbd><kbd>→</kbd>|  
|选择微调大小10px|<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>SHIFT</kbd> + <kbd>↑</kbd><kbd>↓</kbd><kbd>←</kbd><kbd>→</kbd>|  
|选择自动调整大小|<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>0</kbd>|  
|平移画布（手形工具）|按住<kbd>SPACE BAR</kbd>键并选中拖动画布|  
|置于顶层|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>↑</kbd>|  
|置于底层|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>↓</kbd>|  
|前移一层|<kbd>CTRL</kbd> + <kbd>↑</kbd>|  
|下移一层|<kbd>CTRL</kbd> + <kbd>↓</kbd>|  
|复制|按住<kbd>ALT</kbd>键并拖动|  

|编辑/历史记录   |         |  
|:---------|:------------|  
|撤销最后一条命令|<kbd>CTRL</kbd> + <kbd>Z</kbd>|  
|恢复最后一条命令|<kbd>CTRL</kbd> + <kbd>Y</kbd>或者<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>Z</kbd>|  
|剪切|<kbd>CTRL</kbd> + <kbd>X</kbd>或者<kbd>SHIFT</kbd> + <kbd>DELETE</kbd>|  
|复制|<kbd>CTRL</kbd> + <kbd>C</kbd>或者<kbd>CTRL</kbd> + <kbd>INSERT</kbd>|  
|粘贴|<kbd>CTRL</kbd> + <kbd>V</kbd>或者<kbd>SHIFT</kbd> + <kbd>INSERT</kbd>|  
|原位粘贴|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>V</kbd>|  
|删除|<kbd>DELETE</kbd>或者<kbd>BACKSPACE</kbd>|  
|复制选定的控件|<kbd>CTRL</kbd> + <kbd>D</kbd>|  
|加锁|<kbd>CTRL</kbd> + <kbd>2</kbd>|  
|解锁|<kbd>CTRL</kbd> + <kbd>3</kbd>|  

|分组    |         |  
|:---------|:------------|

## 文本处理

|||
|:---------|:------------|  
|开始编辑选中控件的文本|<kbd>ENTER</kbd>或者<kbd>F2</kbd>|  
|提交当前文本|<kbd>ENTER</kbd> (单行控件), <kbd>CTRL</kbd> + <kbd>ENTER</kbd> (多行控件)或者单击任意处|  
|取消当前编辑|<kbd>ESC</kbd>|  
|增加文本大小|<kbd>CTRL</kbd> + <kbd>]</kbd>|  
|减小文本大小|<kbd>CTRL</kbd> + <kbd>[</kbd>|  

## 文本格式

|||
|:---------|:------------|  
|*斜体*|\_下划线\_|  
|[链接]()|[带括号的文本]|  
|[与目标链接]()|[带括号的文本] (样品名称)或[带括号的文本] (网站网址)|  
|**加粗**|\*星号\*|  
|不可见的|-连字符-|  
|删除线|~波浪线~|  
|文字大小|{大小:12} 文本 {大小}|  
|颜色 (十六进制值或颜色名称)|{颜色:#ff0000} 文本 {颜色} 或者 {颜色:红色} 文本 {颜色}|  
|要在控件中打出文本格式的 *, , − -, or [] 需要在它们的前面添加一个 \ (反斜杠)，即： \*, \, −\-, \, [\|  

## 特殊的文本宏命令  

|||
|:---------|:------------|  
|文本排版生成器|在段落或文本区域输入 *lorem*。尝试用 *lorem* 得到变化。|  
|显示当前模型的名称|在标签、文本、目录或二级目录控件中输入 {mockup-name}。|  

## 快速添加

|||
|:---------|:------------|   
|关注快速添加|<kbd>/</kbd>(斜杠)或者<kbd>+</kbd>(加号)|  
|浏览建议列表|<kbd>↑</kbd><kbd>↓</kbd>|  
|关闭建议列表|<kbd>ESC</kbd>|  
|返回到画布|<kbd>ESC</kbd>|  
|在光标处添加控件|<kbd>SHIFT</kbd> + <kbd>ENTER</kbd>|  

## 模型文件处理  

|||
|:---------|:------------|  
|新的空白模型|<kbd>CTRL</kbd> + <kbd>N</kbd>|  
|复制当前模型|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>N</kbd>|  
|新项目|<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>N</kbd>|  
|打开项目|<kbd>CTRL</kbd> + <kbd>O</kbd>|  
|保存为...|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>S</kbd>|  
|给该模型重命名|<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>R</kbd>|  
|关闭当前项目|<kbd>CTRL</kbd> + <kbd>W</kbd>或者<kbd>CTRL</kbd> + <kbd>F4</kbd>|  
|导出该模型的JSON|<kbd>CTRL</kbd> + <kbd>E</kbd>|  
|导入该模型的JSON|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>E</kbd>|  
|导出快照到剪贴板|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>C</kbd>|  
|导出快照为PNG格式|<kbd>CTRL</kbd> + <kbd>R</kbd>|  
|将所有快照导出为PNG格式|<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>R</kbd>|  
|导出为PDF格式|<kbd>CTRL</kbd> + <kbd>P</kbd>|  
|退出Balsamiq Mockups|<kbd>CTRL</kbd> + <kbd>Q</kbd>|  
|浏览模型中打开的桌面文件|<kbd>CTRL</kbd> + <kbd>Page Down</kbd>以及<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>Page Up</kbd>|  

## 视图  

|||
|:---------|:------------|  
|隐藏/显示用户界面库|<kbd>CTRL</kbd> + <kbd>L</kbd>或者<kbd>CTRL</kbd> + <kbd>F1</kbd>|  
|隐藏/显示检查栏|<kbd>CTRL</kbd> + <kbd>;</kbd>或者<kbd>ALT</kbd> + <kbd>ENTER</kbd>|
